# DELTARUNE Thai localization
ยินดีต้อนรับสู่ DELTARUNE รุ่นปรับให้เหมาะกับท้องถิ่นไทย ซึ่งเป็นการแปลภาษา DELTARUNE แบบรู้บริบทและทำให้ตรงตามเจตนาต้นฉบับให้มากที่สุดเท่าที่เป็นไปได้

# การติดตั้ง
หากคุณเป็นผู้ใช้งานธรรมดาที่พยายามติดตั้งแพตช์นี้ ยินดีด้วย แพตช์นี้ยังไม่พร้อมใช้งาน ฉะนั้นรอไปก่อนเด้อ

# โครงสร้างไดเรกทอรี
เนื่องจากการแก้ไขเกม DELTARUNE เป็นอะไรที่ซับซ้อนมากมาย เพื่อเป็นการจัดระเบียบไฟล์ให้เรียบร้อยและทำให้สามารถขยาย DELTARUNE เพิ่มเติมได้เมื่อมีการอัปเดตใหม่ จึงมีไดเรกทอรีและไฟล์ดังนี้
```
@ root directory
/modifications/    # โคดทั้งหมดที่ถูกแก้ไข
    /all-chapters/    # แก้ทุกแช็ปเตอร์
    /chapter1/        # self-explanatory
    /chapter2/        # "------------"
    /chapter3/        # "------------"
    /chapter4/        # "------------"
    /chapter-select/  # "------------"
/scripts/          # ไดเรกทอรีสคริปต์ของ UndertaleModTool ที่ต้องใช้ในการแพตช์
/sprites/          # ไดเรกทอรีสไปรต์ทั้งหมด
main.csx           # ไฟล์หลักในการแพตช์
ump.csx            # Framework ที่ใช้จัดการ; อยู่ภายใต้สัญญาอนุญาต GNU LGPLv2.1
```
